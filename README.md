## typescript 常用命令
```
// 安装(更新)最新版
npm install -g typescript

// //这是一个typescript的交互式控制台，可以用来调试ts脚本，不然只能调试编译后的js
npm install -g ts-node   

// 版本
tsc --version

// 项目初始化
npm --init

// 创建tsconfig.json
tsc --init

// 安装@types来管理类型定义
npm install @types/node --dev-save

```
